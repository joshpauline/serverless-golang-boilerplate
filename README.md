# Serverless Golang Boilerplate

Boilerplate serverless application using AWS API Gateway.

## Installation

Install node dependencies.

```bash
yarn
```

## Usage

Local Invocation

```bash
yarn start
```

Build

```bash
make
```

Deploy

```bash
yarn deploy:{env}
```
